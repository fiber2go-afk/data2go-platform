
# Data2Go™ Net-Zero AI Infrastructure Digital Twin — DCII Prototype

This is a runnable Python/Streamlit prototype for demonstrating the Data2Go concept to the Data Center Innovation Initiative (DCII).

## What the prototype demonstrates

The app simulates and compares data center infrastructure architectures before construction, including:

- Traditional hyperscale architecture
- Distributed fiber-ring architecture
- Hybrid architecture
- Net-zero emissions pathway
- On-site power generation
- Renewables and BESS
- Carbon capture, CO2 pipeline, and injection pathway
- Water sustainability
- AI workload distribution concept
- Climate resiliency
- Community benefit optimization

## How to run

1. Install Python 3.10+.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
streamlit run app.py
```

## Prototype notes

This is not an engineering-grade model yet. It is an operative demonstration version intended to show the software concept, user workflow, and scoring methodology.

For a formal DCII application, the assumptions should be replaced or calibrated with validated datasets:

- ERCOT and utility datasets
- EIA emissions data
- Fiber GIS data
- Water availability and produced-water datasets
- Climate hazard datasets
- CCUS pipeline and sequestration data
- County-level tax and workforce datasets

## Suggested DCII positioning

Data2Go is not proposed as a data center development project. It is proposed as a technology platform that enables data centers to achieve lower emissions, lower resource consumption, greater resiliency, and improved community outcomes before construction begins.
